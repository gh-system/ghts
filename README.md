ghts
====

GHTS : GH 프로그램 매매 시스템.


*********************************************************

- lib : 공용 기능.
- xing : Xing API 관련 일부 주문 및 조회 기능을 소켓 메시지를 통해서 제공
- xing/c32 : xing모듈에서 DLL을 호출하는 부분만 별도로 분리한 32비트 전용 모듈.
- xing/common : 32비트 모듈과 일반 모듈 공용 자료형.

*********************************************************
