ghts
====

GHTS : GH 프로그램 매매 시스템.


*********************************************************

- lib : 공용 기능.
- xing/base : Xing API c32/go 공용 자료형.
- xing/c32 : Xing API DLL을 호출하는 32비트 모듈.
- xing/go : Xing API 기능 호출 (32/64비트 모두 가능)

*********************************************************
